Option A: Manual delivery (simplest)

After payment, you email the buyer a download link (Google Drive/OneDrive) or the file as an attachment.

  1. Buyer completes payment.
  2. You receive order notification (email/WhatsApp/PayFast notice).
  3. Send the buyer the file + quick setup instructions.
Tip: Keep a standard email template ready with the “Enable Macros / Unblock” steps.

Option B: Automatic delivery (recommended for scale)

Use a checkout platform that automatically delivers the file after payment.

  • Payhip / Gumroad (digital product delivery)
  • WooCommerce / Shopify (if you want a full store)
  • PayFast can be added later with a serverless function to unlock downloads
