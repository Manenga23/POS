# Excel POS Template — Netlify Website (Brand-neutral)

This folder is a ready-to-deploy static website you can upload to **Netlify** to sell your Excel POS template.

## What you get
- Landing page (features, pricing, FAQs)
- Documentation page
- Download / delivery page
- License, Terms, Privacy pages
- Netlify contact form (no backend required)
- Security + cache headers in `netlify.toml`

## Deploy to Netlify (no code)
1. Zip the folder (or drag the folder)
2. Netlify → **Add new site** → **Deploy manually**
3. Drag-and-drop the folder
4. Your site goes live with a Netlify URL

## Add your branding
Open and edit:
- `index.html`
- `download.html`
- `privacy.html`
- `terms.html`
- `license.html`

Search for: `[` and fill placeholders like:
- `[YOUR BRAND / STORE NAME]`
- `[YOUR EMAIL]`
- `[YOUR WHATSAPP / PHONE]`

## How buyers get the file
This template ships with a delivery page (`/download`) that explains how you deliver the file.

To make it **automatic**, choose one method:
- Payhip / Gumroad (simple digital download)
- PayFast (can be added later via a simple checkout page)

If you tell me which payment method you want, I can update the site accordingly.

## Forms
The contact form on the home page uses Netlify Forms. After deploying:
- Netlify dashboard → Site → Forms → you’ll see submissions.
